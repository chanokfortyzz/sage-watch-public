# sage-watch-public

Redacted public summary for Sage local workflow state.

This repository intentionally contains summaries only. It must not contain raw queue files, logs, secrets, private task bodies, or local path trees.

Files:
- `status-latest.json`: current redacted machine-readable status
- `status-latest.md`: current redacted human-readable status
- `events.jsonl`: append-only redacted actionable snapshots
